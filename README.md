REST API для справочника организаций с геопоиском.

## Быстрый старт
```bash
git clone https://github.com/reydiamond/Luna-test-case.git
cd Luna-test-case
make env
make up
make migrate
make seed
```

Документация: http://localhost:8000/docs

## API Endpoints

### Организации
- `GET /api/organisations` - список с фильтрами
  - `?building_id={uuid}` - по зданию
  - `?activity_id={uuid}` - по деятельности (рекурсивно)
- `GET /api/organisations/search?name={text}` - поиск по названию
- `GET /api/organisations/nearby?latitude={lat}&longitude={lon}&radius_km={km}` - геопоиск
- `GET /api/organisations/{id}` - детальная информация

### Аутентификация
Все запросы требуют заголовок: `X-API-Key: my_test_api_key`
API ключ можно заменить в .env после его инициализации (по умолчанию: admin)
